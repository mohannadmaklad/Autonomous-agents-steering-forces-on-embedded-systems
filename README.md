# Autonomous-agents-steering-forces-on-embedded-systems
This is an implementation of the concept of autonomous agents &amp; steering forces implemented on a small embedded system.  
Parts used in the project :  
1- STM32F103C8T6  
2-NOKIA LCD 5110  
  
# connections :                                                                                                                        

RST   -----> PA8  
CE    -----> PB12  
DC    -----> PA9  
DIN   -----> PB15  
CLK   -----> PB13  
VCC   -----> 3.3v  
LIGHT -----> GND  
GND   -----> GND
